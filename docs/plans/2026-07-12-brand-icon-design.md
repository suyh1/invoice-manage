# Invoice OCR 品牌图标设计

## 背景

项目当前没有 favicon 或应用图标，浏览器因此显示默认灰色地球。登录页和工作台侧栏也只有 `Invoice OCR` 文字，没有统一的图形品牌标记。

## 设计目标

- 直接表达“发票识别与归档”的核心业务。
- 在 16px 浏览器标签图标中仍能辨认，不依赖文字或细线细节。
- 延续项目现有青绿色主色与克制、专业的财务工作台风格。
- 浏览器标签、移动端快捷方式和页面内品牌区域使用同一个视觉母版。

## 已确认方案

采用“发票 + 扫描框”图形：青绿色方形底，中间为白色折角票据，四角用扫描定位线形成 OCR 语义。图形保持平面、无渐变、无阴影，圆角与项目现有 8px 视觉语言一致。

主色使用项目登录页已有的 `#177c78`。白色票据保留少量青绿色结构线，在小尺寸下仍形成清晰轮廓。图标不使用字母 `IO`，避免在 favicon 尺寸下产生可读性问题。

## 接入范围

- 新增可缩放 SVG favicon，并从同一母版导出 PNG 应用图标。
- 在 HTML 中声明 favicon、Apple Touch Icon、主题色和 Web App manifest。
- 登录页品牌区和工作台侧栏复用一个 React `BrandMark` 组件。
- 窄屏侧栏使用图形标记替代当前纯文字 `IO`。

## 验证标准

- 页面源码能正确引用全部图标资产，资源可由 Vite 构建输出。
- 登录页和工作台品牌区显示同一标记，装饰图形不干扰无障碍名称。
- 16px、32px、180px 等常见尺寸无裁切，票据和扫描框仍有清晰对比。
- TypeScript 检查、相关组件测试和生产构建通过。
