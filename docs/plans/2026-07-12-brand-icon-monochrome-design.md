# Invoice OCR 黑白品牌图标设计

## 背景

项目当前的活跃视觉系统已经转为黑白编辑台风格：登录页以 `#0a0a0a`、白色和少量状态绿色构成，登录后的工作台同样以 `#0a0a0a` 和中性灰为主。现有品牌图标使用旧色 `#177c78`，与页面主题不一致。

## 已确认方案

保留“发票 + 扫描框”的图形结构，只调整颜色：

- 图标底色：`#0a0a0a`
- 发票主体：`#ffffff`
- 扫描定位框：`#dededb`
- 发票折角：`#ececea`
- 发票结构线：`#0a0a0a`

浏览器 `theme-color` 和 Web App manifest 同步使用 `#0a0a0a`。SVG、React 内联图形和所有 PNG 导出必须来自同一配色母版。

## 范围

- 修改 SVG favicon 和 React `BrandMark` 的颜色。
- 重新导出 32、180、192、512 像素 PNG 图标。
- 更新 HTML、manifest 和测试中的主题色。
- 不修改旧登录样式中遗留的 `--auth-accent: #177c78`；该清理不属于本次图标修正。

## 验证标准

- 品牌图标相关文件不再包含 `#177c78`。
- PNG 尺寸正确且视觉保持清晰。
- 相关测试、完整测试和生产构建通过。
- 桌面与移动视口下图标与现有黑白主题一致且无布局变化。
